# Dialog 对话框

在保留当前页面状态的情况下，告知用户并承载相关操作。。

# 基本用法

打开弹框 test
<template>
  <div>
    <hx-button @click.native="dialogVisible = !dialogVisible">打开弹框</hx-button>
    <hx-dialog
        title="提示"
        :visible.sync="dialogVisible"
    >
      <span>test</span>
    </hx-dialog>
  </div>
</template>

<script>
export default {
  name: "dialog-demo",
  data() {
    return {
      dialogVisible: false
    }
  }
}
</script>
