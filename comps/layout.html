# Layout 布局

通过基础的 24 分栏，迅速简便地创建布局。

WARNING

Composition API在vuepress里貌似有点问题，故效果无法展示

# 基础布局

 
<template>
 <div>
     <hx-row class="layout-demo-row">
         <hx-col>
             <div style="background-color: #909399" class="layout-demo-col"></div>
         </hx-col>
     </hx-row>
     <hx-row class="layout-demo-row">
         <hx-col :span="12"><div style="background-color: #909399" class="layout-demo-col"></div></hx-col>
         <hx-col :span="12"><div style="background-color: #e8ebef" class="layout-demo-col"></div></hx-col>
     </hx-row>
     <hx-row class="layout-demo-row">
         <hx-col :span="8"><div style="background-color: #909399" class="layout-demo-col"></div></hx-col>
         <hx-col :span="8"><div style="background-color: #e8ebef" class="layout-demo-col"></div></hx-col>
         <hx-col :span="8"><div style="background-color: #66a1f1" class="layout-demo-col"></div></hx-col>
     </hx-row>
     <hx-row class="layout-demo-row">
         <hx-col :span="6"><div style="background-color: #909399" class="layout-demo-col"></div></hx-col>
         <hx-col :span="6"><div style="background-color: #e8ebef" class="layout-demo-col"></div></hx-col>
         <hx-col :span="6"><div style="background-color: #66a1f1" class="layout-demo-col"></div></hx-col>
         <hx-col :span="6"><div style="background-color: #0964dc" class="layout-demo-col"></div></hx-col>
     </hx-row>
 </div>
</template>
<style>
.layout-demo-row {
    margin-bottom: 20px;
}
.layout-demo-col {
    min-height: 30px;
    border-radius: 4px;
}
</style>

# 分栏布局

 
<template>
    <hx-row class="layout-demo-row" :gutter="20">
        <hx-col :span="6"><div style="background-color: #909399" class="layout-demo-col"></div></hx-col>
        <hx-col :span="6"><div style="background-color: #e8ebef" class="layout-demo-col"></div></hx-col>
        <hx-col :span="6"><div style="background-color: #66a1f1" class="layout-demo-col"></div></hx-col>
        <hx-col :span="6"><div style="background-color: #0964dc" class="layout-demo-col"></div></hx-col>
    </hx-row>
</template>
<style>
    .layout-demo-row {
        margin-bottom: 20px;
    }
    .layout-demo-col {
        min-height: 30px;
        border-radius: 4px;
    }
</style>

# 混合布局

 
<template>
    <hx-row class="layout-demo-row" :gutter="20">
        <hx-col :span="3"><div style="background-color: #909399" class="layout-demo-col"></div></hx-col>
        <hx-col :span="5"><div style="background-color: #e8ebef" class="layout-demo-col"></div></hx-col>
        <hx-col :span="7"><div style="background-color: #66a1f1" class="layout-demo-col"></div></hx-col>
        <hx-col :span="9"><div style="background-color: #0964dc" class="layout-demo-col"></div></hx-col>
    </hx-row>
</template>
<style>
    .layout-demo-row {
        margin-bottom: 20px;
    }
    .layout-demo-col {
        min-height: 30px;
        border-radius: 4px;
    }
</style>

# Row Attributes

参数 说明 类型 可选值 默认值
gutter 栅格间隔 number — 0

# Col Attributes

参数 说明 类型 可选值 默认值
span 栅格占据的列数 number — 24
