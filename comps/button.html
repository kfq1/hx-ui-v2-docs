# Button 按钮

常用的操作按钮。

# 基本用法

默认按钮 主要按钮 成功按钮 信息按钮 警告按钮 危险按钮
<template>
    <div style="display: flex; justify-content: space-around">
        <hx-button >默认按钮</hx-button>
        <hx-button type="primary">主要按钮</hx-button>
        <hx-button type="success">成功按钮</hx-button>
        <hx-button type="info">信息按钮</hx-button>
        <hx-button type="warning">警告按钮</hx-button>
        <hx-button type="danger">危险按钮</hx-button>
    </div>
</template>
