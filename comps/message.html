# Message 消息提示

常用于主动操作后的反馈提示。。

# 基础用法

showMsg showVnodeMsg
<template>
   <div>
       <hx-button @click.native="showMsg">showMsg</hx-button>
       <hx-button @click.native="showVnode">showVnodeMsg</hx-button>
   </div>
</template>
<script>
    export default {
        name: "message-demo",
        methods: {     
            showMsg() {
                this.$message({
                    message: 'msgTest'
                })
            },
            showVnode() {
                const h = this.$createElement;
                this.$message({
                    message: h('p', null, 'Vnode')
                });
            },
        }
    }
</script>

# 不同状态

消息 成功 失败 警告
<template>
    <div>
        <hx-button @click.native="open1">消息</hx-button>
        <hx-button @click.native="open2">成功</hx-button>
        <hx-button @click.native="open3">失败</hx-button>
        <hx-button @click="open4">警告</hx-button>
    </div>
</template>
<script>
    export default {
        name: "message-demo",
        methods: {
            open1() {
                this.$message('这是一条消息提示');
            },
            open2() {
                this.$message({
                    message: '恭喜你，这是一条成功消息',
                    type: 'success'
                });
            },

            open4() {
                this.$message({
                    message: '警告哦，这是一条警告消息',
                    type: 'warning'
                });
            },

            open3() {
                this.$message.error('错了哦，这是一条错误消息');
            }
        }
    }
</script>

# Options

参数 说明 类型 可选值 默认值
message 消息文字 string / VNode — —
type 主题 string success/warning/info/error info

# 方法

调用 Message 或 this.$message 会返回当前 Message 的实例。如果需要手动关闭实例，可以调用它的 close 方法。

方法名 说明
close 关闭当前的 Message
