# TablePro 表格

用于展示多条结构类似的数据

WARNING

table-pro采用了了粘性定位，解决了老表格节点过多问题，并且内部表格渲染默认使用了virtualTable优化性能，但由于 vuepress无法使用Composition API,故md无法展示效果，项目里可正常使用

# 基本用法

123
 

<template>
    <div>
        <table-pro :data="data">
            <hx-table-column-input field="id" title="ID"/>
            <hx-table-column-input field="id" title="IDleft" fixed="left"/>
            <hx-table-column-input field="name" title="姓名" width="300" fixed="left">
                <template>
                    <div>123</div>
                </template>
                <template v-slot:title>
                    slot姓名
                </template>
            </hx-table-column-input>
            <hx-table-column-input field="sex" title="性别"/>
            <hx-table-column-input field="age" title="年龄"/>
            <hx-table-column-input v-for="item in 100" field="name" :title="'姓名' + item" :key="item"/>
            <hx-table-column-input field="id" title="fixedRight" fixed="right"/>
        </table-pro>
    </div>
</template>

<script>
    export default {
        name: "table-pro-demo",
