# Input 输入框

通过鼠标或键盘输入字符

# 基本用法

2 
<template>
  <div>
      {{value}}
      <hx-input v-model="value"/>
  </div>
</template>
<script>
    export default {
        name: "input-demo",
        data() {
            return {
                value: 2
            }
        }
    }
</script>
<style>
.hx-input {
    width: 180px;
}
</style>
