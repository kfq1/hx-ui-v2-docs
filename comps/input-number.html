# InputNumbe 计数器

仅允许输入标准的数字值，可定义范围

WARNING

input-number采用自定义指令的方式给dom添加事件，由于vuepress的影响，打包过程会出现问题。 所以文档里的实例只有展示作用，正常项目使用组件是没问题的

# 基本用法

<template>
    <hx-input-number v-model="value"></hx-input-number>
</template>
<script>
    export default {
        name: "input-number-demo",
        data() {
            return {
                value: 2
            }
        }
    }
</script>

# 步数

1 1
<template>
    <div>
        <div>{{value1}} {{ value2}}</div>
        <hx-input-number v-model="value1" :step="2"></hx-input-number>
        <hx-input-number v-model="value2" :step="0.5"></hx-input-number>
    </div>
</template>
<script>
    export default {
        name: "input-number-demo",
        data() {
            return {
                value1: 1,
                value2: 1,
            }
        }
    }
</script>

# 最大、最小值

    <hx-input-number v-model="value" :max="10" :min="-5"></hx-input-number>
</template>
<script>
    export default {
        name: "input-number-demo",
        data() {
            return {
                value: 1

# Attributes

参数 说明 类型 可选值 默认值
value / v-model 绑定值 number — 0
min 设置计数器允许的最小值 number — -Infinity
max 设置计数器允许的最大值 number — Infinity
step 计数器步长 number — 1
disabled 是否禁用计数器 boolean — false
